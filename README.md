## 📌 Overview

This repository showcases my learning and hands-on implementation during a **3-Day MERN Stack Workshop** conducted at my university.

The workshop focused on building a **full-stack application** using modern technologies, covering backend development, frontend integration, and deployment.

---

## 🧠 Key Learnings

* Backend development using Node.js & Express.js
* Database handling with MongoDB (Compass & Shell)
* Building RESTful APIs
* Frontend development with React.js
* Full-stack integration (Frontend ↔ Backend ↔ Database)
* Real-world features like file upload and email services
* Deployment of MERN stack application
